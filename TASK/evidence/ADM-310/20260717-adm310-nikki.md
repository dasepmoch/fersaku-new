## 3. Backend contract (consumed)

| Endpoint | Permission | Use |
| --- | --- | --- |
| `GET /v1/admin/withdrawals` | `withdrawals.review` | Bounded list (FE AdminWithdrawal shape via AdminRead) |
| `GET /v1/admin/withdrawals/{id}` | `withdrawals.review` | Detail |
| `POST /v1/admin/withdrawals/{id}/review` | `withdrawals.review` | Body `{ action: approve\|hold\|reject, reason }`; approve may disburse |

Domain sources: `adminRead` (reads), `adminWrite` (commands).

Disbursement callback remains server/webhook (`INT-180`); FE does not drive provider callbacks.

### Slash contract

- OpenAPI + FE list: `/v1/admin/withdrawals` (no trailing slash)
- Router: dual mounts — AdminRead `Get("/withdrawals")` under `/v1/admin` and WithdrawalService `Get("/")` under `/v1/admin/withdrawals` both serve list
- Unit test locks FE path string
