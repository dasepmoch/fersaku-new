### Avatar disposition (INT-175)

- **DISABLED / OUT-OF-SCOPE** — no upload, no objects endpoints, no localStorage media.
- Profile uses display name only (no avatar chrome added).

### Bank security

- List DTO exposes `accountNumberMasked` only; mappers reject full number leakage.
- Full `accountNumber` is write-only on create/update body; never in query keys.
- Mutations set `requireRecentMfa: true` for sensitive bank ops.
- Invalidate `bankAccounts` + `withdrawalLock` after bank mutations.

### Removed on API path

- `fersaku-seller-profile-settings` localStorage read/write
- `appendClientAuditEvent` fake audit append
- Hardcoded mock bank as API truth
