## 3. Endpoints

| Method | Path | Permission | Notes |
| --- | --- | --- | --- |
| GET | `/v1/admin/kyc` | `kyc.review` | Bounded queue (`limit` ≤200 default 50); optional `status`, `age=30m\|2h` |
| GET | `/v1/admin/kyc/{caseId}` | `kyc.review` | Case + document metadata + transitions; no raw R2 key |
| POST | `/v1/admin/kyc/{caseId}/transition` | `kyc.review` | Body: `action` + `reason`; reject/clarify require reason |
| GET | `/v1/admin/kyc/{caseId}/documents/{documentId}/content` | `kyc.review` + recent authentication `kyc.document.view` | Server decrypt stream; `X-Audit-Reason` ≥12; `Cache-Control: no-store, private` |

### Transition actions (BE)

`START_REVIEW` | `VENDOR_CHECK` | `NEEDS_CLARIFICATION` | `APPROVE` | `REJECT` | `EXPIRE`

| UI column control | Action |

## 4. Behavior

| Surface | Transport | Notes |
| --- | --- | --- |
| Board (api) | GET `/v1/admin/kyc` | Bounded first result; client search/age filter |
| Detail (api) | GET case when dialog opens from queue row (metadata on list + detail mapper) | Documents are metadata only |
| Transition (api) | POST transition | Idempotency-Key + audit reason; approve requires recent authentication when available |
| Document view (api) | GET content binary | auth purpose `kyc.document.view`; blob URL memory 2m TTL; revoke on hidden/unmount |
| Mock | seed | `apiKycSeed` board; mock transition/local override; mock 1×1 PNG view |
| Credential routes | unchanged | Still BE `kyc.review` hazard — ADM-200 registry split remains follow-up |

### PII / secret policy

- List/detail schemas reject `storageKey`, `downloadUrl`, `presignedUrl`, etc.
- Query keys: `["admin","kyc","bounded",filters]` / `["admin","kyc","case",caseId]` only.
- Decrypted bytes: `apiBinaryRequest` → component memory only; never React Query.
- Response headers: `no-store`, `nosniff`, CSP sandbox, safe `Content-Disposition`.
- Audit log: case/document/merchant/actor/reason length/size — never plaintext.
