- Browser total never sent as authority (mapper omits total/gross/unitPrice).
- Server `amount`/`gross` from create response mapped into memory model only.
- Continue (details→qris) remains presentational — does **not** create intent.

- `createIdempotencyIntentHolder()` mints opaque UUID on first pay.
- Double-click: `createPendingDedupe` + `paying` disable CTA.
- Retry/timeout recovery reuses same key; body change → local conflict (no auto-rotate).
