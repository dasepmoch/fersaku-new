## 3. Policies (summary)

### Query

- Keys: tenant/`storeId` already on seller keys; no secrets in keys (`queryKeyLooksSensitive`)
- `keepPrevious: true` / `withKeepPreviousData()` for filter/cursor smoothness
- `queryFn(signal)` always; React Query aborts prior fetch on key change
- Stale: public 60s · private 30s · finance 15s · secret/auth 0
- Safe GET retry only; mutations never

### Mutation

- `MUTATION_RETRY = false` on hook + provider
- `createIdempotencyKey()` / `createIdempotencyIntentHolder()` — same key on double-click/manual retry; `reset()` only for new intent
- `createPendingDedupe()` disables concurrent CTA
- Money/admin/secret: no optimistic (documented)

### Checkout fix

```diff
- idempotencyKey: `checkout_${product.id}_${email}`
+ idempotencyKey: idempotencyRef.current.getKey() // opaque UUID
```
