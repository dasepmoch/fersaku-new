### Secret policy

- List/GET schemas exclude `signingSecret`.
- Query keys never include claim/token/secret material.
- Claim result must not be written to React Query cache.
- Reveal helper: 5m TTL, clear on unmount + `visibilitychange` hidden.

### Pagination

- Route has no `TablePagination`; BE returns bounded first page (≤50).
- No new paging control added (disposition: bounded first result).
