## 3. Backend contract (consumed)

| Endpoint | Permission | Use |
| --- | --- | --- |
| `GET /v1/admin/audit-logs` | `audit.read` | Search (`action`, `resourceType`, `resourceId`, `actorUserId`, `limit`) |
| `GET /v1/admin/audit-logs/{eventId}` | `audit.read` | Detail |
| `GET /v1/admin/audit-integrity` | `audit.read` | JCS-1 verifier status |
| `POST /v1/admin/audit-exports` | `audit.read` | Async export job + reason |
| `GET /v1/admin/audit-exports/{exportId}` | `audit.read` | Job poll |

Domain source: `adminRead`. List response shape: `{ data: { items }, meta }`.
