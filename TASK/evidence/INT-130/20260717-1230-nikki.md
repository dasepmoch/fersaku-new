## 1. Chosen threat model

**Pattern: session-bound CSRF with authenticated re-issuance (task option 2; hash-at-rest double-submit).**

| Asset | Control |
| --- | --- |
| Session cookie | HttpOnly, Secure (prod), SameSite=Lax (Strict optional admin) |
| CSRF proof | Random 32-byte token; **only hash** stored on `auth_sessions.csrf_token_hash` |
| Client store | **Process memory only** (`shared/api/csrf.ts`) — never localStorage/sessionStorage/URL |
| Transport | Same-origin browser `apiRequest` (`credentials: "include"`) + header `X-CSRF-Token` |
| Enforcement | CSRF middleware only when **valid** `SessionMeta` present (resolved session) |
| Stale cookie | Unresolved cookie does **not** require CSRF → anonymous login/magic-link/password/logout can replace cookie |
| Hard refresh | `GET /v1/auth/session` **rotates** CSRF hash and returns new raw once |
| Recovery | On `403 AUTH_CSRF_INVALID`, FE `recoverCsrfOnce` + `withCsrfRecovery` retries **once** with **same idempotency key** |
| Logs | Reporter redacts keys matching `token`/`csrf`/… |

Not chosen: non-HttpOnly CSRF cookie (option 1). Reason: existing BE already stores hash only; re-issue on safe GET matches hard-refresh without a readable cookie surface.

## 3. Paths supported by design

1. **Login → mutation:** login body `csrfToken` → `setCsrfToken` → unsafe methods auto-header via hooks.
2. **Hard refresh → mutation:** memory empty → `ensureCsrfToken` → `GET /v1/auth/session` rotates → mutation with new token.
3. **Stale CSRF mid-session:** mutation `AUTH_CSRF_INVALID` → `withCsrfRecovery` once → same `Idempotency-Key`.
4. **Missing/invalid token (valid session):** `403 AUTH_CSRF_INVALID`.

## 6. Consumer notes (INT-120+)

```ts
import {
 setCsrfToken,
 clearCsrfToken,
 wireHttpClientCsrfHooks,
 withCsrfRecovery,
} from "@/shared/api/csrf";

wireHttpClientCsrfHooks();
// after login/magic-link:
setCsrfToken(data.csrfToken);
// mutations that must survive one CSRF rotation:
await withCsrfRecovery(() =>
 apiRequest(path, { method: "POST", idempotencyKey: key, ... }),
);
// logout / session expired:
clearCsrfToken();
```
