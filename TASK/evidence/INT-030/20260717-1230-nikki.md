## 2. Topology (summary)

See [topology.md](./topology.md).

- Browser: `fetch("/v1/...", { credentials: "include" })` — no absolute public API required.
- Next: rewrite `/v1` and `/v1/:path*` → `${API_INTERNAL_URL origin}/v1/...` (default host `http://127.0.0.1:18080`).
- SSR: `getApiInternalUrl()` / `requireApiInternalUrl()` server-only (INT-110 will consume).
- `NEXT_PUBLIC_API_URL` deprecated for browser same-origin; absolute mode remains escape hatch.
