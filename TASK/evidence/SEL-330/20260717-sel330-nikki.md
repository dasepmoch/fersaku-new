- List/GET schemas exclude `apiKey` / `claimToken`.
- Query keys never include claim/token/auth/secret material.
- Claim result must not be written to React Query cache.
- Reveal helper: 5m TTL, clear on unmount + `visibilitychange` hidden.
- Separate from `signingSecret` (SEL-320).
