## 3. Endpoints

| Method | Path | Notes |
| --- | --- | --- |
| GET | `/v1/stores/{storeId}/reviews` | Bounded first result (limit ≤50); filter q/status/rating; membership; foreign → 404 |
| GET | `/v1/stores/{storeId}/reviews/summary` | Published store aggregate (count/avg/rating1..5) |
| PUT | `/v1/stores/{storeId}/reviews/{reviewId}/reply` | Create/update single public reply; version conflict 409 |
| POST | `/v1/stores/{storeId}/reviews/{reviewId}/report` | Reason enum + context; dedupe conflict; no moderation status change |

Permissions: `seller.store.read` (list/summary), `seller.store.write` (reply/report). Platform admin bypass membership check (same pattern as SEL-260).
