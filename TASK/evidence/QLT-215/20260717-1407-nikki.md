## 2. How to run

```bash
# 1) Disposable backend stack: deps → migrate → QLT-110 seed → api/worker ready
npm run test:e2e:api:stack

# 2a) Full: Playwright starts Next (DATA_SOURCE=api) + runs harness
npm run test:e2e:api

# 2b) Backend-only probes (no Next webServer)
E2E_API_SKIP_WEBSERVER=1 E2E_API_HAS_NEXT=0 \
  PLAYWRIGHT_API_BASE_URL=http://127.0.0.1:18080 \
  npm run test:e2e:api

# 2c) Stack + tests (CI-shaped)
npm run test:e2e:api:ci

# Mock e2e unchanged
npm run test:e2e:smoke
```

Env knobs:

| Env | Purpose |
| --- | --- |
| `API_INTERNAL_URL` | Go API origin (default `http://127.0.0.1:18080`) |
| `MAILPIT_URL` | Mailpit UI/API (default `http://127.0.0.1:8025`) |
| `E2E_API_SKIP_WEBSERVER=1` | Do not start Next from Playwright |
| `E2E_API_HAS_NEXT=0\|1` | Skip/run same-origin rewrite probes |
| `E2E_API_TEARDOWN=1` | `compose down` on stack script exit |
| `E2E_API_SKIP_SEED=1` | Skip QLT-110 seed |
| `PLAYWRIGHT_API_PORT` | Next port (default `3120`) |
