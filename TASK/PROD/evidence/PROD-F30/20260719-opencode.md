## Balance path (documentation)

1. Paid order → payment capture → ledger credit (available or pending per product rules).
2. Seller balance = ledger projection for merchant.
3. Withdrawal: quote → reserve → disburse (Xendit) → webhook complete/fail (idempotent).
4. No double payout on disburse retry (unit).
