## Secure cookie policy (code + local evidence)

`AuthHandler.cookieSecure` (`backend/internal/adapters/http/handlers/auth.go`):

- `SecureCookies` config true on staging/production, **or**
- request TLS, **or**
- `X-Forwarded-Proto: https`, **or**
- `Origin` / `Referer` starts with `https://`

Cookies: `HttpOnly`, `SameSite=Lax` default (`SameSiteStrict` optional).

Integration: `TestIdentityRegisterVerifyLoginSessionLogout` asserts **HttpOnly** + non-empty `csrfToken`.
